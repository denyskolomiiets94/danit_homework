Поясніть роль і значення кожного з наступних компонентів в архітектурі ОС Linux:

a) Kernel - це ядро Linux, яке працює по типу як посередник між залізом (процесор, оперативкой, hdd/ssd).
Також ядро дозволяє працювати з іншими пристроями за допомогою драйверів.
Ядро Linux також вирішує яка програма і коли отримує доступ до CPU, тобто керую процесами.
Ядро керує пам'яттю, розподіляє між програмами.
Працює з файловими системами ( доступ до файлів, підтримка різного формату (ext4,NTFS,FAT32.. ) )
Робота з мережею ( TCP/IP, механізм маршрутизації, фаєрвол)
Також працює з безпекою ( контроль прав доступу (користувачі, групи ).


b) Libraries - це вже готовий збірник з якихось методів, або функція.
наприклад хочу ввести якусь команду зі словом "time", но я не знаю яку саме або що можна зробити зі словом "time",
то можна подивитись по готовій збірці команд, треба ввести команду "men -k time" і буде доступна велика кількість
команд, що включають слово "time".


c) System utilities - це інструмент для роботи з Linux.
ls - la - показує що є в директорії
cp - копіює файл
mv - перейменовує
rm - видалити
mkdir - створює папку
touch file.txt - створити текстовий файл з іменем файл
htop - диспетчер задач, показує завантаження оперативки, процесора і показує самі процеси
df - показує місце на диску
chmod - змінює права
...

System utilities - це базові інструменти Linux для роботи з системою через термінал.
