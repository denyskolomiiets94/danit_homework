1) зайти в Git Bash і створити директорію  з назвою "PetClinic"
2) добавити в директорію  "PetClinic" два файла Vagrantfile і .env
3) створити папку в директорії  "PetClinic" з назвою "provision" та перейти в неї 
4) добавити в папку "provision" три файли app.sh ,  postgres.sh , petclinic.service
5) повернутись на початкову директорію  "PetClinic"
Має бути так:  в директорії "PetClinic" має бути 2 файла (Vagrantfile і .env)  + одна папка ( provision) ,
в папці  "provision" має бути три файла (app.sh ,  postgres.sh , petclinic.service)
6) запустити команду vagrant up 
7) першим буде створюватись db-vm, треба буде обрати доступний мережевий інтерфейс мосту ( обрати WiFi адаптер )
і дочекатись завершення
8) другим буде створюватись app-vm, треба буде обрати доступний мережевий інтерфейс мосту ( обрати WiFi адаптер )
і дочекатись завершення
9) в браузері можна буде перейти за посиланням http://192.168.1.114:8080
